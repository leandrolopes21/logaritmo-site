Logaritmo

"O logaritmo é uma ferramenta matemática que nos ajuda a descobrir qual expoente precisamos elevar uma base para obter um determinado número"

Entendendo o conceito

Imagine a seguinte pergunta: "A que potência devemos elevar 2 para obter 8?".

A resposta é 3, porque 2³ = 2x2x2 = 8

Em termos de logaritmo, essa pergunta seria escrita assim: log2 8 = 3

Isso se lê como "logaritmo de 8 na base 2 é igual a 3".

Os Elementos do Logaritmo

    Vamos detalhar os componentes:

  • Base (b): É o número que está sendo elevado a alguma potência. No exemplo 〖log〗_2 8=3, a base é 2.
  • Logaritmando (x): É o número cujo logaritmo estamos calculando. No exemplo 〖log〗_2 8=3, o logaritmando é 8.
  • Logaritmo (y): É o resultado, ou seja, o expoente ao qual a base deve ser elevada para obter o logaritmando. No exemplo 〖log〗_2 8=3, o logaritmo é 3.

Podemos generalizar a relação entre potências e logaritmos da seguinte forma:

Se b^y=x, então log⁡〖b_x=y〗
